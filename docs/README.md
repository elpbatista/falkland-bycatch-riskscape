# Documentation

This folder contains the current public documentation set for the reusable
riskscape workflow.

- `authentication.md`: provider credential setup.
- `datasets.md`: source dataset references and access notes.
- `workflow.md`: current high-level workflow map.
- `script_inventory.md`: scripts intended for public documentation.
- `publishing_checklist.md`: publishing-readiness decisions and open items.
- `code_style_guide.md`: coding conventions; still under review for public
  release.

Older proposal notes, experimental reports, and outdated implementation memos
have been removed from the public docs tree.
