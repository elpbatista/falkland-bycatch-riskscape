# Documentation

This folder contains the public documentation set for the Falkland Bycatch
Riskscape workflow. The notebooks provide the main guided walkthrough; these
Markdown files are shorter references for setup, data restoration, scripts,
workflow stages, and release decisions.

- `authentication.md`: provider credential setup.
- `datasets.md`: source dataset references and access notes.
- `workflow.md`: current high-level workflow map.
- `script_inventory.md`: current script roles and cleanup status.
- `notebooks.md`: public notebook roles and replacements for older notebooks.
- `zenodo_manifest.md`: released Zenodo data-bundle structure.
- `publishing_checklist.md`: release history and post-release maintenance
  notes.
- `data_folder_assessment.md`: release-time data audit snapshot.
- `code_style_guide.md`: lightweight coding conventions.

Older proposal notes, experimental reports, and outdated implementation memos
have been removed from the public docs tree.
